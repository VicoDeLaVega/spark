🎯 Stratégie d'entretien Nvidia / AMD / Intel

  • Nvidia : Focus sur CUDA, warp divergence, occupancy, Tensor Cores, cuDNN/cuBLAS pipeline
  • AMD : ROCm/HIP, wavefronts (= warps), LDS (= shared mem), architecture CDNA/RDNA
  • Intel : oneAPI/SYCL, EU (Execution Units), subgroups, XMX pour AI, Xe HPG vs HPC
  • Connaître Amdahl's Law, Roofline Model, memory coalescing par cœur
  • Savoir expliquer le trade-off occupancy vs. register pressure et quand chaque stratégie gagne
