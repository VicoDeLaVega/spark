11.1 Définition du Scan (Prefix Sum)

Le scan (inclusive) calcule pour chaque position i la somme de tous les éléments de 0 à i. C'est une primitive fondamentale pour les algorithmes parallèles (allocation dynamique, compaction, radix sort...).

Inclusive vs Exclusive Scan

  • Inclusive : output[i] = input[0] + input[1] + ... + input[i]
  • Exclusive : output[i] = input[0] + input[1] + ... + input[i-1] (output[0] = 0)

Exemple : [3, 1, 7, 0, 4, 1, 6, 3]
Inclusive scan : [3, 4, 11, 11, 15, 16, 22, 25]
Exclusive scan : [0, 3, 4, 11, 11, 15, 16, 22]
